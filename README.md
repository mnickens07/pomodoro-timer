# Pomodoro Timer

This repository contains a simple Pomodoro timer web app.

## Live site

- https://mnickens07.github.io/pomodoro-timer/

## Files

- `index.html` — entry point for GitHub Pages.
- `pomodoro.html` — standalone app file.
- `.nojekyll` — disables Jekyll processing for GitHub Pages.

## Local location

This repo is stored locally at:

- `c:\Users\mnick\pomodoro-temp-repo`

## How to update

1. Edit `index.html` or `pomodoro.html`.
2. Commit your changes:
   ```bash
   git add .
   git commit -m "Update Pomodoro app"
   git push origin master
   ```

## Notes

- The live app is served from the `master` branch.
- If you want the root URL to remain the app, keep `index.html` in the repo root.
